# Portfolio Net - Portfolio Profissional

Bem-vindo ao repositório do **Portfolio Net**, um template de portfólio moderno, minimalista e responsivo desenvolvido com React e Vite.

## ✨ Destaques
- **Design Editorial**: Estética limpa focada em tipografia e imagens de alta qualidade.
- **Carrosséis Interativos**: Habilidades e Trabalhos exibidos de forma dinâmica.
- **Páginas de Detalhes**: Cada projeto possui sua própria página dedicada para estudos de caso.
- **Totalmente Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.

## 📁 Documentação
